Entity Relationship Diagram

Relasi Antar Entitas

Produk ke Order_Item (One-to-Many)
Satu Produk dapat menjadi bagian dari banyak Order_Item (Sebuah produk bisa dipesan dalam banyak pesanan).
Order_Item akan memiliki ID Produk sebagai foreign key untuk merujuk ke Produk.

Order ke Order_Item (One-to-Many)
Satu Order dapat berisi banyak Order_Item (Satu pesanan bisa mencakup banyak produk).
Order_Item akan memiliki ID Order sebagai foreign key untuk menghubungkan ke Order.

Pelanggan ke Order (One-to-Many)
Satu Pelanggan dapat melakukan banyak Order.
Order akan memiliki ID Pelanggan sebagai foreign key untuk merujuk ke Pelanggan.

Admin ke Produk (One-to-Many)
Satu Admin dapat mengelola banyak Produk.
Produk akan memiliki ID Admin sebagai foreign key untuk menunjukkan admin yang mengelola produk.

What (Apa?)

Website ini memberikan informasi mengenai produk atau barang yang tersedia di Giri Store. Pengguna bisa melihat detail produk seperti nama, deskripsi, harga, dan gambar. Pemesanan barang dilakukan dengan menghubungi toko melalui WhatsApp.

Who (Siapa?)

Pengguna atau pelanggan yang mengunjungi website ini untuk mencari informasi barang. Admin atau pemilik toko yang mengelola informasi barang di website.

When (Kapan?)

Website ini dapat diakses kapan saja oleh pengguna untuk melihat informasi barang. Pemesanan barang dapat dilakukan kapan saja melalui WhatsApp.

Where (Di mana?)

Website ini dapat diakses secara online di alamat https://giri-store.vercel.app/.

Why (Mengapa?)

Tujuan website ini adalah untuk memberikan informasi tentang produk yang dijual dan memfasilitasi pemesanan melalui WhatsApp tanpa proses checkout langsung di website.

How (Bagaimana?)

Pengguna mengunjungi website, melihat informasi produk, dan jika tertarik, mereka menghubungi Giri Store melalui link WhatsApp yang disediakan.
